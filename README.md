# Django ToDo App

This is a simple ToDo web application built with Django.

## Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Das-art/todo.git
Navigate to the Project Directory:

cd todo

Create a Virtual Environment:

python -m venv venv

Activate the Virtual Environment:
On Windows:

venv\Scripts\activate


source venv/bin/activate
Install Dependencies:

pip install -r requirements.txt
Run
Start the Development Server:

python manage.py runserver
Access the Application:
Open a web browser and navigate to http://127.0.0.1:8000/.
Test
Run Unit Tests:

python manage.py test

Run Integration Tests:

python manage.py test todo.tests.integration
Additional Information
Database Setup:

Modify the DATABASES settings in settings.py to configure the database as needed.
Environment Variables:

Set environment variables for sensitive information such as database credentials.
Deployment:

Deploy the project to a production server following best practices.
